gagepy.rdb¶

Module containing functionality that reads and parses tab-delimited formatted timeseries files from the United States Geological Survey (USGS) National Water Information System (NWIS). The tab-delimited data files are also referred to as “rdb” format. Please see http://pubs.usgs.gov/of/2003/ofr03123/6.4rdb_format.pdf for more information regarding the tab-delimited (rdb) format.

authors:2016 by Jeremiah Lant, see AUTHORS
license:United States Geological Survey (USGS), see LICENSE file

gagepy.rdb.parse_parameter_code(match)[source]¶

Process a regular expression match for a parameter code and a description. The parameter code and description string line is made up of 4 parts:

  1. a data descriptor value
  2. a parameter code value
  3. a parameter statistic value (may or may not exist)
  4. a parameter description
Parameters:match (_sre.SRE_Match) – a regular expression match object
Returns:parameter code, statistic (if it exists), and description
Return type:tuple 
>>> import re
>>> parameter_code_re = "(#)\D+([0-9]{2})\D+([0-9]{5})(\D+[0-9]{5})?(.+)"
>>> line = "#       06      00060   00003   Discharge, cubic feet per second (Mean)"
>>> match = re.search(rdb.parameter_code_re, line)
>>> parameter_code = rdb.parse_parameter_code(match)
>>> parameter_code
('06_00060_00003', 'Discharge, cubic feet per second (Mean)')

class gagepy.usgsgage.USGSGage(name, parameters)[source]¶

A class that contains data and functionality of a United States Geological Survey (USGS) gage.

Parameters:
  • name (string) – Name of USGS gage
  • parameters (list) – List of Parameter objects that contain data and functionality of a timeseries data parameter.

See also

Parameter

>>> import numpy as np
>>> from datetime import datetime
>>> name = "USGS 03290500 KENTUCKY RIVER AT LOCK 2 AT LOCKPORT, KY"
>>> dates = np.array([
...     datetime(2015, 8, 1, 0, 0),
...     datetime(2015, 8, 2, 0, 0),
...     datetime(2015, 8, 3, 0, 0),
...     datetime(2015, 8, 4, 0, 0),
...     datetime(2015, 8, 5, 0, 0)])
>>> parameters = [
...    Parameter(name = "Discharge",
...              dates = dates,
...              values = np.array([100, 110, 105, 107, 112]),
...              units = "cubic feet per second (Mean)",
...              code = "06_00060_00003"),
...    Parameter(name = "Stage",
...              dates = dates,
...              values = np.array([10, 20, 25, 15, 5]),
...              units = "feet",
...              code = "03_00065")
... ]
>>> usgs_gage = USGSGage(name, parameters)
>>> usgs_gage.name
USGS 03290500 KENTUCKY RIVER AT LOCK 2 AT LOCKPORT, KY

gagepy.output¶

Module of classes that contains data and functionality to create various output views of data from a United States Geological Survey (USGS) gage. The outputs consist of data summaries in restructured text format and html format. In addition, plots of the data can be output in json objects to be embedded in webpages.

authors:2016 by Jeremiah Lant, see AUTHORS
license:United States Geological Survey (USGS), see LICENSE file

Add a new ending to a filename,

Parameters:
  • file (string) – File or path to file
  • suffix (string) – Suffix to add to end of file
  • ext (string) – File extension
  • delimiter (string) – Delimiter, default is the dash character
Returns:

New file
Return type:

string

Note

Spaces in filenames are replaced by delimiter to keep with Unix file naming conventions.

>>> import utils
>>> utils.add_ending(file="dv.txt", suffix="summary", ext=".txt")
'dv-summary.txt'
>>> utils.add_ending(file="dv.rdb", suffix="summary", ext=".rst", delimiter="_")
'dv_summary.rst'
>>> utils.add_ending(file="/home/play/dv.rdb", suffix="summary", ext=".rst")
'/home/play/dv-summary.rst'
>>> utils.add_ending(file="daily values.rdb", suffix="summary", ext=".rst")
'daily-values-summary.rst'
